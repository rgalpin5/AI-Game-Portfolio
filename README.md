## Prerequisites

## How to Run

Each game is in its own directory with its own instructions

### Tetris:
    1. Navigate to project directory
    2. Run the Main Script ('python Tetris.py' in the dir)
    3. (Optional) Run a pre-trained model
        i. On line 74 of Agent.py, change the name to the name of desired model
    4. To train a new model, comment out the loading methods in the Agent.py file
